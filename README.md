# customer-churn-analysis-ml
Customer Churn Analysis and Prediction using Python, Data Visualization, and Machine Learning algorithms including Linear Regression, Logistic Regression, Decision Tree, and Random Forest.

Customer Churn Analysis & Prediction

This project focuses on analyzing customer churn data and building machine learning models to predict customer churn behavior.

The notebook includes:

Data loading and preprocessing
Data filtering and analysis
Exploratory Data Analysis (EDA)
Data visualization
Machine Learning model implementation
Model evaluation

The project uses multiple ML algorithms to compare prediction performance.
